Это программа состоит из 3 модулей:
	server.py - сервер, куда передаются данные и откуда можно получить
	статистику посещений вашего сайта.
	    -адрес сервера/api/stat?site=SITE, где SITE это адрес вашего сайта
	
	app.py - приложение, которое реализовано на сервере.
	С его помощью можно получать статистику посещений.
	
	database.py - файл, в котором реализован класс с базой данных.
	
Чтобы использовать счетчик посещений на своем сайте скачайте архив, и
запустите файл server.py, затем добавьте в html своего сайта картинку и
загружайте ее с адреса  localhost:8080/api/picture?id=<имя вашего сайта>

Чтобы получить статистику посещений вашего сайта перейдите по адресу
localhost:8080/api/stat?id=<имя вашего сайта>

Чтобы добавить картинку с количеством посещений на ваш сайт необходимо
после запуска server.py встроить на ваш сайт следующий url:
localhost:8080/api/stat?site=<имя вашего сайта>

В файле requirements.txt находятся модули, которые необходимо установить
перед началом работы с сервером.

В директории tests/ находятся модульные тесты для этой программы.
